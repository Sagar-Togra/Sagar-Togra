Event Handlers

From 11 to 20;

11. Aux Click [ auxclick ]

An auxclick event is when the user has used a non primary mouse button.

In the below example when you right click the box it will execute the sayHello function, Try yourself

For Example
<span onauxclick="sayHello();"></span>

Press your right mouse button to see the function works

If you want to see the difference between auxclick and click event. Click here

12. On Context Menu [ oncontextmenu ]

The element with a "oncontextmenu" attribute, will create an event when rigth mouse button is pressed over the element and will execute whatever the function has been given to it.;

For Example
<span oncontextmenu="sayHello();"></span>

13. On DBL Click [ ondblclick ]

The element with a "ondblclick" attribute, will create an event when left mouse button is pressed two times over the element and will execute whatever the function has been given to it.;

For Example
<span ondblclick="sayHello();"></span>

14. On Blur [ onblur ]

The element with a "onblur" attribute, will create an event when mouse button is released over the element and will execute whatever the function has been given to it.;

For Example
<span onblur="sayHello();"></span>
