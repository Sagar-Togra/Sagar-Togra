CSS Height Property

In this page you will get to know about CSS Height Property, how it works and all its values.

Definition

It sets the height of an element.
It doesn't include padding, margin, border.

Syntax

height : auto | length | initial | inherit;

Lets know about each value one by one

  1. auto  :  The browser calculates the height.
  2. length  :  Two types of length Absolute and Relative
    1. Absolute
      • cm  :  In centimeters
      • mm  :  In milimeters
      • in  :  In inches
      • px  :  In pixels (1px = 1/96th in)
      • pt  :  In points (1pt = 1/72th in)
      • pc  :  In picas (1pc = 12pt)
    2. Relative
      • em  -  Relative to the font size of letter M
      • ex  :  Relative to the height of "x".
      • rem  :  Relative to the font size of root element
      • ch  :  Relative to the width of zero "0"
      • vmin  :  Relative to the smallest dimension of the viewport in percentage
      • vmax  :  Relative to the largest dimension of the viewport in percentage
      • vw  :  Relative to the width of the viewport in percentage
      • vh  :  Relative to the height of the viewport
      • %  :  Relative to the parent element in percentage.
  3. inital  :  It sets the property value to its default value
  4. inherit  :  It sets the property value same as that of its parent element.
