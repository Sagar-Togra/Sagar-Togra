Event Handlers

From 1 to 10;

Mouse related events;

1. On Click [ onclick ]

The element with a "onclick" attribute, when clicked will create an event and will execute whatever the function has been given to it.;

For Example
<span onclick="sayHello();"></span>

2. On Mouse Over [ onmouseover ]

The element with a "onmouseover" attribute, will create an event when mouse comes over the element and will execute whatever the function has been given to it.;

For Example
<span onmouseover="sayHello();"></span>

3. On Mouse Down [ onmousedown ]

The element with a "onmousedown" attribute, will create an event when mouse button is pressed down over the element and will execute whatever the function has been given to it.;

For Example
<span onmousedown="sayHello();"></span>

4. On Mouse Up [ onmouseup ]

The element with a "onmouseup" attribute, will create an event when mouse button is released over the element and will execute whatever the function has been given to it.;

For Example
<span onmouseup="sayHello();"></span>

5. On Mouse Out [ onmouseout ]

The element with a "onmouseout" attribute, will create an event when mouse cursor lefts the element and will execute whatever the function has been given to it.;

For Example
<span onmouseout="sayHello();"></span>

6. On Mouse Move [ onmousemove ]

The element with a "onmousemove" attribute, will create an event when mouse cursor moves on the element and will execute whatever the function has been given to it.;

For Example
<span onmousemove="sayHello();"></span>

7. On Mouse Enter [ onmouseenter ]

The element with a "onmouseenter" attribute, will create an event when mouse cursor enters on a element and will execute whatever the function has been given to it.;

For Example
<span onmouseenter="sayHello();"></span>

8. On Mouse Leave [ onmouseleave ]

The element with a "onmouseleave" attribute, will create an event when mouse cursor leaves the element and will execute whatever the function has been given to it.;

For Example
<span onmouseleave="sayHello();"></span>

9. On Mouse Wheel [ onmousewheel ]

The element with a "onmousewheel" attribute, will create an event when mouse wheel is used on the element and will execute whatever the function has been given to it.;

For Example
<span onmousewheel="sayHello();"></span>

10. On Scroll [ onscroll ]

The element with a "onscroll" attribute, will create an event when mouse wheel is used on the element and will execute whatever the function has been given to it.;

For Example
<span onscroll="sayHello();"></span>
