CSS Margin Property

Definition

It specifies all of the four margin of an element.

You can specify margin of an element using one of the four ways

1. margin  :  10px 20px 30px 40px;
  • Top margin is 10px.
  • Right margin is 200px.
  • Bottom margin is 70px.
  • Left margin is 90px.
This div has above margin values
2. margin  :  10px 20px 30px;
  • Top margin is 30px.
  • Left and Right margin is 200px.
  • Bottom margin is 10px.
This div has above margin values
3. margin  :  10px 20px;
  • Top and Bottom margin is 50px.
  • Right and Left margin is 100px.
This div has above margin values
4. margin  :  10px;
  • All four margin are 50px;
This div has above margin values

Note. -ve values are allowed.

Syntax

margin   :   length | % | auto | initial | inherit ;

Lets learn about every value one by one
